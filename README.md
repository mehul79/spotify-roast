
# Spotify Roaster

A fun and interactive project that generates humorous roasts based on your Spotify listening habits.

## Features

- Analyze your top tracks and artists
- Generate personalized roasts about your music taste
- Share roasts with friends
- Discover insights about your listening patterns

## Getting Started

### Prerequisites
- node.js installed
- Spotify API credentials

### Installation

```bash
git clone <repository-url>
cd spotify_roast
npm i
npm run dev
```

### Configuration

1. Create a `.env` file with your Spotify API keys
2. Run the application
